# Torres Compra Cash - Ready to Launch

## Fastest way to launch today

### Option A: Vercel (easiest for this package)
1. Create a GitHub account if you do not have one.
2. Upload this folder to a new GitHub repository.
3. Go to Vercel and import the repository.
4. Click Deploy.
5. Your site will be live on a temporary `.vercel.app` URL.

## Before you launch
Open `app/page.tsx` and replace:
- `email`
- `phoneDisplay`
- `phoneDigits`
- `messengerUrl`
- `areas`
- `formAction`

## Formspree setup
1. Create a Formspree account.
2. Create a new form.
3. Copy the form endpoint.
4. Replace `https://formspree.io/f/your-form-id` in `app/page.tsx`.

## Run locally
```bash
npm install
npm run dev
```

## Domain later
After deployment, connect your custom domain in Vercel.
